# ProPlanter 🌱

A browser-based two-player educational board game that simulates plant growth
through 5 stages across 50 tiles, with 4 difficulty levels (Easy, Medium, Hard,
and Tournament Mode).

## Tech Stack
- HTML5
- CSS3 (custom animations, glassmorphism UI, responsive design)
- Vanilla JavaScript (no frameworks)
- Progressive Web App (PWA) — installable, offline-capable via Service Worker

## Features
- 5-stage plant growth simulation (Pot → Soil → Seed → Manure → Grow)
- 4 game modes with distinct risk/reward mechanics, including a Tournament
  Mode with power cards (Shield, Loan, Precision, Swap)
- Dynamic weather events in Hard Mode (Heatwave, Flood, Pests, Wind, Frost)
- In-game quiz system and educational content
- Persistent leaderboard with score tracking
- Sound effects and background music
- Fully animated, responsive UI with a dark ambient theme

## How to Run
1. Clone this repository
2. Open `index.html` in a browser, or serve the folder with a local server
   (e.g. VS Code Live Server) for full PWA functionality
